# Repo Energy Transition Sector Source Map

Use this file as the repo-native entry point for `energy-transition-sector-pack`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Sector market, grid, resource, contracting, financing, and ESG rules.
4. Skill-local trigger cases and reference notes.
5. Prior sector-pack outputs only as context.

If the question is about project finance, use `project-finance-structurer`. If it is about transition-risk exposure, use `scenario-stress-testing`.

## Usage notes

- Use the source map before treating the sector thesis as settled.
- Keep resource, grid, contract, and finance evidence separate.
- Do not reduce transition complexity to a single score.
