# Repo Food Agriculture Sector Source Map

Use this file as the repo-native entry point for `food-agriculture-sector-pack`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Sector market, supply chain, food security, and climate rules.
4. Skill-local trigger cases and reference notes.
5. Prior sector-pack outputs only as context.

If the question is about industrial capability, use `industrialization-hilirisasi-analyzer`. If it is about socioeconomic effects, use `socioeconomic-multiplier-modeler`.

## Usage notes

- Use the source map before drawing a food-security conclusion.
- Keep supply, price, resilience, and capability separate.
- Do not confuse harvest volume with national security.
