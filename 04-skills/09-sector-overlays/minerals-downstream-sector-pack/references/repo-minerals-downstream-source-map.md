# Repo Minerals Downstream Sector Source Map

Use this file as the repo-native entry point for `minerals-downstream-sector-pack`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Sector market, downstreaming, industrial policy, and ESG rules.
4. Skill-local trigger cases and reference notes.
5. Prior sector-pack outputs only as context.

If the question is about industrial capability, use `industrialization-hilirisasi-analyzer`. If it is about strategic priority, use `national-priority-alignment`.

## Usage notes

- Use the source map before calling a downstream asset strategic.
- Keep ore, processing, market access, and policy support separate.
- Do not confuse domestic processing with domestic value capture.
