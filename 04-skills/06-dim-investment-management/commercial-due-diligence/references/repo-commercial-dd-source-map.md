# Repo Commercial Due Diligence Source Map

Use this file as the repo-native entry point for `commercial-due-diligence`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Commercial diligence, valuation, and decision-quality protocols.
4. Owner-certified internal records and primary transaction or project documents.
5. Prior diligence only as context.

If the question is whether the opportunity is worth pursuing, use `investment-thesis-builder`. If the question is whether the pack is ready for decision, use `decision-quality-gate`.

## Usage notes

- Use the source map before summarizing commercial diligence.
- Keep diligence separate from mandate, approval, and audit packaging.
- Do not hide weak economics behind positive narrative.
