# Repo Financial DD Source Map

Use this file as the repo-native entry point for `financial-due-diligence`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Financial diligence, valuation, downside, and assurance protocols.
4. Owner-certified internal records and primary transaction or project documents.
5. Prior diligence only as context.

If the question is whether the opportunity is in mandate, use `investment-mandate-interpreter`. If it is whether the pack is ready for decision, use `decision-quality-gate`.

## Usage notes

- Use the source map before summarizing financial diligence.
- Keep diligence separate from mandate, approval, and audit packaging.
- Do not hide weak economics behind polished spreadsheets.
