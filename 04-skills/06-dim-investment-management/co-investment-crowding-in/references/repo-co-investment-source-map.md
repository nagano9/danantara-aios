# Repo Co-Investment Source Map

Use this file as the repo-native entry point for `co-investment-crowding-in`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Co-investment, additionality, crowding-in, and impact protocols.
4. Owner-certified internal records and primary transaction or project documents.
5. Prior co-investment assessments only as context.

If the question is whether the opportunity is in mandate, use `investment-mandate-interpreter`. If it is whether the investment is worth pursuing, use `investment-thesis-builder`.

## Usage notes

- Use the source map before analyzing crowding-in or co-investment.
- Keep additionality separate from mandate and approval questions.
- Do not count capital mobilization without evidence of attribution.
