# Repo Investment Thesis Source Map

Use this file as the repo-native entry point for `investment-thesis-builder`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Thesis, valuation, downside, impact, and approval protocols.
4. Owner-certified internal records and primary transaction or project documents.
5. Prior theses only as context.

If the question is whether the opportunity is in mandate, use `investment-mandate-interpreter`. If it is whether the pack is ready for decision, use `decision-quality-gate`.

## Usage notes

- Use the source map before building a thesis.
- Keep thesis separate from mandate and approval.
- Do not remove downside or dissent to make the thesis clean.
