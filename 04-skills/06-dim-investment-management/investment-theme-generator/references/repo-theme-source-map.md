# Repo Theme Source Map

Use this file as the repo-native entry point for `investment-theme-generator`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Investment themes, mandate, strategy, and portfolio protocols.
4. Owner-certified internal records and primary transaction or project documents.
5. Prior themes only as context.

If the question is whether the opportunity is in mandate, use `investment-mandate-interpreter`. If it is whether the opportunity is worth pursuing, use `investment-thesis-builder`.

## Usage notes

- Use the source map before proposing a theme.
- Keep themes separate from mandate and approval.
- Do not invent a theme to justify an asset already chosen.
