# Repo Prudence Source Map

Use this file as the repo-native entry point for `prudence-and-downside-gate`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Downside, loss, and prudence thresholds.
4. Skill-local trigger cases and reference notes.
5. Prior prudence outputs only as context.

If the question is about loss tolerance or downside risk, use `long-term-value-test`. If it is about decision robustness, use `decision-quality-gate`.

## Usage notes

- Use the source map before recommending a course with material downside.
- Keep prudence separate from fear-driven conservatism.
- Do not optimize for speed when loss asymmetry is material.
