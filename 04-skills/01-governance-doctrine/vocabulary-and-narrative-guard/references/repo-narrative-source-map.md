# Repo Vocabulary and Narrative Source Map

Use this file as the repo-native entry point for `vocabulary-and-narrative-guard`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Approved vocabulary, narrative discipline, and external tone constraints.
4. Skill-local trigger cases and reference notes.
5. Prior narrative outputs only as context.

If the question is about safe wording, use `transparency-disclosure-gate`. If the issue is about certainty or overclaiming, use `danantara-way-governance`.

## Usage notes

- Use the source map before finalizing public-facing text.
- Keep tone aligned with the institution, not with personal style.
- Do not let persuasive language outrun verified substance.
