# Repo Executive Board Brief Source Map

Use this file as the repo-native entry point for `executive-board-brief-builder`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved board and governance artifacts.
3. Executive briefing rules, decision framing, and concise evidence.
4. Skill-local trigger cases and reference notes.
5. Prior board briefs only as context.

If the question is about disclosure, use `transparency-disclosure-gate`. If it is about strategic alignment, use `strategic-national-interest-test`.

## Usage notes

- Use the source map before compressing analysis into a board-ready brief.
- Keep facts, assumptions, and recommendations visibly separated.
- Do not sacrifice traceability for elegance.
