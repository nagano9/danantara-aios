# Repo Public Accountability Source Map

Use this file as the repo-native entry point for `public-accountability-packager`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Accountability, disclosure, and public communication rules.
4. Skill-local trigger cases and reference notes.
5. Prior accountability outputs only as context.

If the question is about what may be disclosed, use `transparency-disclosure-gate`. If it is about tone or framing, use `vocabulary-and-narrative-guard`.

## Usage notes

- Use the source map before translating analysis into public messaging.
- Keep accountability, disclosure, and persuasion separate.
- Do not overstate certainty to improve optics.
