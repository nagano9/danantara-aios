# Repo National Priority Alignment Source Map

Use this file as the repo-native entry point for `national-priority-alignment`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. National priorities, strategic alignment, and policy fit.
4. Skill-local trigger cases and reference notes.
5. Prior alignment outputs only as context.

If the question is about strategic public interest, use `strategic-national-interest-test`. If it is about value and downside, use `long-term-value-test`.

## Usage notes

- Use the source map before claiming something matches a national priority.
- Keep policy fit separate from implementation feasibility.
- Do not assume alignment just because the label sounds strategic.
