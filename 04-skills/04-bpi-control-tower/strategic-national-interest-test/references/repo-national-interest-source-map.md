# Repo Strategic National Interest Source Map

Use this file as the repo-native entry point for `strategic-national-interest-test`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Strategic public-interest, national value, and downside rules.
4. Skill-local trigger cases and reference notes.
5. Prior test outputs only as context.

If the question is about long-term value or downside, use `long-term-value-test`. If it is about public accountability, use `public-accountability-packager`.

## Usage notes

- Use the source map before calling something nationally beneficial.
- Keep national value separate from implementation popularity.
- Do not compress a contested policy judgment into a slogan.
