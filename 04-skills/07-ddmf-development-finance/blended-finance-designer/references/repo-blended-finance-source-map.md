# Repo Blended Finance Source Map

Use this file as the repo-native entry point for `blended-finance-designer`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Concessionality, risk layering, and blended capital design rules.
4. Skill-local trigger cases and reference notes.
5. Prior financing designs only as context.

If the question is about viability gaps, use `viability-gap-support-assessor`. If it is about project structure, use `project-finance-structurer`.

## Usage notes

- Use the source map before mixing concessional and commercial capital.
- Keep subsidy, return, and crowding-in separate.
- Do not disguise support as market pricing.
