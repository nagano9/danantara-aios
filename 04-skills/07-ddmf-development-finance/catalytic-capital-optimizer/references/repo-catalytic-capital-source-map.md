# Repo Catalytic Capital Source Map

Use this file as the repo-native entry point for `catalytic-capital-optimizer`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Capital leverage, crowding-in, and minimum intervention rules.
4. Skill-local trigger cases and reference notes.
5. Prior catalytic outputs only as context.

If the question is about bankability or viability gaps, use `bankability-gap-analyzer` or `viability-gap-support-assessor`. If it is about impact, use `development-impact-assessor`.

## Usage notes

- Use the source map before defining the minimum viable intervention.
- Keep leverage, additionality, and subsidy separate.
- Do not make catalytic capital a synonym for generous capital.
