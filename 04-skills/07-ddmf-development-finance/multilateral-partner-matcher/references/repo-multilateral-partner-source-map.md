# Repo Multilateral Partner Source Map

Use this file as the repo-native entry point for `multilateral-partner-matcher`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Partner fit, mandate compatibility, and financing collaboration rules.
4. Skill-local trigger cases and reference notes.
5. Prior partner outputs only as context.

If the question is about financing design, use `blended-finance-designer`. If it is about public outcome, use `development-impact-assessor`.

## Usage notes

- Use the source map before naming a partner as the right partner.
- Keep mandate fit, capacity, and incentives separate.
- Do not assume a famous partner is a fit partner.
