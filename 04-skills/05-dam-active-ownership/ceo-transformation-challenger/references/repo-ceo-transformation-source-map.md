# Repo CEO Transformation Source Map

Use this file as the repo-native entry point for `ceo-transformation-challenger`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Leadership accountability, performance change, and intervention rules.
4. Skill-local trigger cases and reference notes.
5. Prior challenge outputs only as context.

If the question is about talent or governance redesign, use `management-accountability-contract`. If it is about turnaround work, use `turnaround-triage`.

## Usage notes

- Use the source map before challenging a leader or plan.
- Keep diagnosis, challenge, and authority separate.
- Do not turn disagreement into a personal judgment.
