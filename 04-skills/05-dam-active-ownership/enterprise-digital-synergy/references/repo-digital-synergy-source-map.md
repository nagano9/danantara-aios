# Repo Enterprise Digital Synergy Source Map

Use this file as the repo-native entry point for `enterprise-digital-synergy`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Digital capability sharing, platform leverage, and operating model rules.
4. Skill-local trigger cases and reference notes.
5. Prior digital synergy outputs only as context.

If the question is about platform architecture, use `shared-platform-architect`. If it is about operating model change, use `transformation-agenda-orchestrator`.
