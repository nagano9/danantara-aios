# Repo Talent Meritocracy Source Map

Use this file as the repo-native entry point for `talent-meritocracy-assessor`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Talent, role, performance, succession, and merit rules.
4. Skill-local trigger cases and reference notes.
5. Prior talent outputs only as context.

If the question is about workforce planning, use `workforce-planning`. If it is about accountability, use `management-accountability-contract`.

## Usage notes

- Use the source map before rating talent merit.
- Keep performance, potential, and succession separate.
- Do not let tenure substitute for merit or vice versa.
