# Repo Value Creation Source Map

Use this file as the repo-native entry point for `value-creation-plan-builder`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Value creation, performance improvement, and ownership rules.
4. Skill-local trigger cases and reference notes.
5. Prior value-creation outputs only as context.

If the question is about diagnostic root cause, use `financial-performance-diagnostic` or `operational-excellence-diagnostic`. If it is about integration after M&A, use `post-merger-integration-planner`.

## Usage notes

- Use the source map before turning diagnosis into a plan.
- Keep initiatives, owners, milestones, and economics explicit.
- Do not confuse a list of actions with a value plan.
