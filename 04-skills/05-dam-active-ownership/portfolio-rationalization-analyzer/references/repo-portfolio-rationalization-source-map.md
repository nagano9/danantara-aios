# Repo Portfolio Rationalization Source Map

Use this file as the repo-native entry point for `portfolio-rationalization-analyzer`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Portfolio role, ownership value, and rationalization rules.
4. Skill-local trigger cases and reference notes.
5. Prior rationalization outputs only as context.

If the question is about capital allocation, use `capital-allocation-reviewer`. If it is about exit readiness, use `divestment-exit-readiness`.

## Usage notes

- Use the source map before proposing to shrink the portfolio.
- Keep strategic fit, value, and exit options separate.
- Do not rationalize simply because an asset is difficult.
