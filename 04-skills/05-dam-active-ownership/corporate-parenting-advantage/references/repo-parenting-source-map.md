# Repo Corporate Parenting Source Map

Use this file as the repo-native entry point for `corporate-parenting-advantage`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Ownership advantage, active ownership, and stewardship logic.
4. Skill-local trigger cases and reference notes.
5. Prior parenting outputs only as context.

If the question is about accountability or capital discipline, use `capital-allocation-reviewer`. If it is about role or structure, use `management-accountability-contract`.

- Use the source map before arguing that parent ownership adds value.
- Keep advantage, authority, and intervention separate.
- Do not assume ownership alone creates performance.
