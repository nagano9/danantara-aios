# Repo National Champion Competitiveness Source Map

Use this file as the repo-native entry point for `national-champion-competitiveness`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Scale, capability, governance, innovation, and competition rules.
4. Skill-local trigger cases and reference notes.
5. Prior competitiveness outputs only as context.

If the question is about industrial capability, use `industrialization-hilirisasi-analyzer`. If it is about risk or resilience, use `enterprise-risk-aggregator`.

## Usage notes

- Use the source map before calling scale durable.
- Keep capability, market access, governance, and protection separate.
- Do not assume protected scale equals global competitiveness.
