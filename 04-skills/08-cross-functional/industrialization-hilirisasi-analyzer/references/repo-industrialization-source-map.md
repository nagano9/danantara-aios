# Repo Industrialization Hilirisasi Source Map

Use this file as the repo-native entry point for `industrialization-hilirisasi-analyzer`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Industrialization, downstreaming, value-add, and capability rules.
4. Skill-local trigger cases and reference notes.
5. Prior analysis outputs only as context.

If the question is about national priority or strategic fit, use `national-priority-alignment`. If it is about competitiveness, use `national-champion-competitiveness`.

## Usage notes

- Use the source map before calling downstreaming an automatic win.
- Keep policy intent, industrial logic, and execution capacity separate.
- Do not equate local content growth with actual capability.
