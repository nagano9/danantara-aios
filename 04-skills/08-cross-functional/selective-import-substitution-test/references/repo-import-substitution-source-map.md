# Repo Selective Import Substitution Source Map

Use this file as the repo-native entry point for `selective-import-substitution-test`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Capability, supply-chain, competition, and substitution rules.
4. Skill-local trigger cases and reference notes.
5. Prior substitution outputs only as context.

If the question is about industrial capability or downstreaming, use `industrialization-hilirisasi-analyzer`. If it is about competitiveness, use `national-champion-competitiveness`.

## Usage notes

- Use the source map before calling substitution selective.
- Keep import dependence, local capability, and feasibility separate.
- Do not confuse restriction with resilience.
