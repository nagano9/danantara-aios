# Repo Procurement Commercial Source Map

Use this file as the repo-native entry point for `procurement-commercial-strategy`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved Danantara governance artifacts and controlled templates.
3. Sourcing, competition, contracting, incentives, and supplier governance rules.
4. Skill-local trigger cases and reference notes.
5. Prior procurement outputs only as context.

If the question is about legal sufficiency, use `legal-sufficiency-checker`. If it is about risk allocation, use `enterprise-risk-aggregator`.

## Usage notes

- Use the source map before selecting a sourcing strategy.
- Keep competition, lifecycle cost, and capability separate.
- Do not let urgency weaken supplier governance.
