# Repo Evidence Citation Source Map

Use this file as the repo-native entry point for `evidence-citation-auditor`. It is an index, not an authority by itself.

## Source order

1. Law, regulation, official decision, and formal mandate.
2. Approved assurance artifacts and controlled templates.
3. Citation style, source traceability, and evidence linking rules.
4. Skill-local trigger cases and reference notes.
5. Prior citation outputs only as context.

If the question is about source hierarchy, use `source-hierarchy-resolver`. If it is about a decision package, use `audit-trail-packager`.

## Usage notes

- Use the source map before declaring a citation sufficient.
- Keep quote, paraphrase, and inference distinct.
- Do not cite a source in place of a conclusion.
